NuGetPAWS v1.0.0
================
http://chrisvanhoose.wordpress.com/

NuGetPAWS is a web app for tracking your NuGet package usage and recommending upgrades based on author and community recommendations.

Technologies
------------
* ASP.NET MVC 5.1
* EF 6 Code First 
* AutoMapper
* Twitter Bootstrap
* MSTest
* Moq
* Azure Mobile Services
* SQL Server
